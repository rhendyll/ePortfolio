Grade Collector App

Description:

Grade Collector Game is an interactive game where players collect grades and avoid obstacles. The goal is to score as high as possible by collecting positive grades while avoiding obstacles that end the game. Use arrow keys or WASD to move. Collect grades to increase your score. Avoid red obstacles that end the game. Displays the top 5 high scores. Ends when colliding with obstacles or moving out of bounds. Option to restart the game Java Swing for GUI AWT for event handling Timer for game loop and obstacle appearance "How to Play:" Start the game by clicking "Play." Move using the arrow keys or WASD. Collect grades and avoid obstacles. View the leaderboard after the game ends.

Context-Free Grammar to Chomsky Normal Form

Description:

A CFG to CNF converter tool is a software application or online utility designed to transform Context-Free Grammars (CFGs) into Chomsky Normal Form (CNF). CFGs are a formal way to describe the syntax of programming languages and other formal languages, using variables (non-terminal symbols), terminals (symbols of the language), a start symbol, and production rules. CNF is a standardized and simplified form of CFGs where all production rules adhere to specific formats: either a single terminal symbol, or two non-terminal symbols. This conversion is crucial because many parsing algorithms, such as the CYK algorithm, require grammars to be in CNF for efficient processing. A CFG to CNF converter automates the often tedious and error-prone manual conversion process, which involves steps like eliminating epsilon productions, unit productions, and ensuring all remaining productions conform to the CNF format. These tools typically accept a CFG as input (often in a textual or structured format) and output its equivalent CNF representation, simplifying the implementation and analysis of formal languages.
